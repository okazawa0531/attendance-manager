# 操作マニュアル

## ログイン

1. ブラウザでシステムURLを開く
2. メールアドレスとパスワードを入力し「サインイン」をクリック
3. 管理者は管理ページ、一般ユーザーはユーザーページへ自動遷移

---

## 管理者向け操作

### ユーザー追加

1. 管理ページの「ユーザーを追加」ボタンをクリック
2. 以下の情報を入力：
   - **氏名**：フルネーム（例：山田 太郎）
   - **メールアドレス**：ログインに使用するメールアドレス
   - **役割**：一般ユーザー または 管理者
   - **パスワード**：初期パスワード（8文字以上、英大文字・小文字・数字を含む）
3. 「ユーザーを作成」ボタンをクリック
4. 作成後、追加したユーザーにメールアドレスと初期パスワードを別途通知してください

> **注意**: パスワードは8文字以上で、英大文字・英小文字・数字をそれぞれ1文字以上含む必要があります。

### ユーザー削除

1. ユーザー一覧の該当ユーザー行の「削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック

> **注意**: 削除したユーザーの勤怠データは残ります。ユーザー自身は削除されます。

---

## 一般ユーザー向け操作

### 今日の勤怠登録

#### 打刻で登録する方法

1. 「今日の勤怠」ページを開く
2. 出勤時に「出勤打刻」ボタンをクリック → 現在時刻が自動入力される
3. 退勤時に「退勤打刻」ボタンをクリック → 現在時刻が自動入力される
4. 「登録する」ボタンをクリックして保存

#### 手動で時刻を入力する方法

1. 出勤時間・退勤時間の入力欄に直接時刻を入力
2. 必要に応じて休憩時間（分）を変更（デフォルト：60分）
3. 「登録する」ボタンをクリックして保存

#### ステータスの種類

| ステータス | 説明 |
|-----------|------|
| 出勤 | 通常の勤務日 |
| 欠勤 | 無断欠勤・体調不良等 |
| 休日 | 土日・祝日など |
| 有給休暇 | 有給休暇取得日 |

#### 登録済みデータの修正

同じ日に再度「今日の勤怠」ページを開くと既存データが読み込まれます。修正後「更新する」ボタンをクリックしてください。

---

### 勤怠履歴の確認

1. 「勤怠履歴」タブをクリック
2. 年・月を選択すると該当月の勤怠記録が一覧表示される
3. ページ上部に月次サマリー（出勤日数・有給日数・欠勤日数・総勤務時間）が表示される

## よくある質問

**Q: 打刻を忘れた場合はどうすればよいですか？**  
A: 「今日の勤怠」ページで手動入力できます。時刻欄に直接入力して「登録する」をクリックしてください。

**Q: 過去の日付の記録を修正したいのですが？**  
A: 現在のバージョンでは「勤怠履歴」ページから過去の記録の修正はできません。管理者にご連絡ください（将来バージョンで対応予定）。

**Q: パスワードを忘れた場合は？**  
A: 管理者にご連絡ください。管理者がパスワードをリセットします。
